# DCL

→ It is used to control the dataflow between the users.

Types of DCL statements : 

1. GRANT 
2. REVOKE

---

1. GRANT : 
    
    → it is used to give the permission to user.
    
    Syntax : 
    
    ```sql
    GRANT SQL_STATEMENTS
    ON TABLE_NAME
    TO USER_NAME;
    ```
    

---

1. REVOKE : 
    
    → it is used to take back the permission given to the user.
    
    Syntax : 
    
    ```sql
    REVOKE SQL_STATEMENTS 
    ON TABLE_NAME 
    FROM USER_NAME;
    ```
