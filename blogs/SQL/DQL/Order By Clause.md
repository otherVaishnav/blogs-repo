# Order By Clause

→ It order’s the records of result table. 

→ ORDER BY executes after select clause. 

→ we can use col_name, expressions , MRF, alias in ORDER BY 

→ We can also use the position of a column from result table

```sql
eg. 
SELECT ENAME, SAL 
FROM EMP
ORDER BY 2; 
-- order's according to the salary. 
```
