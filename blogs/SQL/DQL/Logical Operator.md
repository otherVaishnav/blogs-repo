---

**Logical Operators** are used to combine multiple conditions in a SQL `WHERE` or `HAVING` clause. They evaluate to TRUE, FALSE, or UNKNOWN.

Key logical operators include:
*   **`AND`**: TRUE if all conditions are TRUE.
*   **`OR`**: TRUE if any of the conditions is TRUE.
*   **`NOT`**: Reverses the value of any boolean operator.

**Syntax Example:**
```sql
SELECT * FROM employees 
WHERE salary > 5000 AND (department = 'IT' OR department = 'HR');
```
