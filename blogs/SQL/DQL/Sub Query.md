# Subqueries

---

A **Subquery** (or Nested Query) is a query placed inside another SQL query. It is used to return data that will be used in the main (outer) query as a condition or value.

*   **Single-Row Subquery**: Returns zero or one row. Used with operators like `=`, `>`, `<`.
*   **Multi-Row Subquery**: Returns multiple rows. Used with operators like `IN`, `ANY`, `ALL`.

**Syntax Example:**
```sql
-- Find employees who earn more than the company average
SELECT name, salary 
FROM employees 
WHERE salary > (SELECT AVG(salary) FROM employees);
```
