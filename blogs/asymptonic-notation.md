→ The efficiency of a program is measured with the help of asymptotic notations.

→ this are mathematical notations used to describe the running time of an algorithm according to the input.

There are mainly three asymptotic notations:

- Big-O notation
    
    ⇒ worst-case complexity of an algorithm.
    ⇒ O(g(n))
    ⇒ it is widely used to analyze an algorithm as we are always interested in the worst-case scenario. 
    
- Omega notation
    
    ⇒ best case complexity of an algorithm. 
    ⇒ eg. Ω(g(n)) 
    
- Theta notation
    
    ⇒ average-case complexity of an algorithm.
    ⇒ Θ(g(n))
