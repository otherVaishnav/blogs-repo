# Linear Search

→ Linear search performance scales linearly with the number of items (n) in the list.

1. Best-Case Complexity: O(1)
2. Worst-Case Complexity: O ( n )
3. Average-Case Complexity : O (n)
    - On average, the element will be somewhere in the middle, requiring roughly n/2 comparisons.

#### When to use Linear Search:

1. When the dataset is **small**.

2. When the dataset is **unsorted.** 

3. When simplicity is preferred over maximum performance. ( you don’t know anything else ).
