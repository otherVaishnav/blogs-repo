# Tree

→ A non-linear data structure.

Types : 

1. Binary tree 
    1. Full Binary Tree
    2. Perfect Binary Tree
    3. Complete Binary Tree
    4. Balanced Binary Tree
    
2. Binary search tree
3. Self Balancing : 
    1. AVL ( Adelson-Velsky and Landis ) tree. 
    2. Red Black tree.
4. Multi way : 
    1. B tree
    2. B+ Tree
5. Specialized : 
    1. Heap 
    2. Trie ( Prefix Tree )
    3. Suffix Tree
    4. Segment Tree
    5. Fenwick Tree (Binary Indexed Tree):
