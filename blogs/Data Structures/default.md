Learning Data Structures is important before we move on to algorithms. 
Data structures helps us to store data in different different ways, we can structure the storage accorgin to our needd. That is the use of data structure. In this module you will be learning about the famous types of data structures which are widely used and asked in the interviews.

### Types of Data Structures:
    Data structures are mainly of two types:
    1. Linear Data Structure
    2. Non-Linear Data Structure

## 1. Linear Data Structure:
    In linear data structure, the data elements are arranged in a sequential manner. If you traverse the data structure from start you will be able to visit all the elements one by one, sequentially. 

    Here are the famous types of linear data structures:
    1. Array
    2. Linked List
    3. Stack
    4. Queue
